I use Flutter for App Development, this is a framework in which you can build fully responsive apps for Android and IOS with only ONE source code. This is excellent in many aspects, for instance I will mention the following advantages:


  • Code maintainance is much more efficient, easy and fast; an update will affect both versions of the app at the same time.
  • Shorter period of time to build two fully responsive apps, one for android and one for IOS.
  • Flutter is a very intuitive programming language, therefore it will be easy to understand code between members if there is a team working on the project.

I have developed two apps with flutter, one is available on the google play store, the other one is on the final stages, it will soon be available in digital stores. Both Apps are for language learning and they have been personal projects.
