This is the first App I build with Flutter. One of my favourite activities is watching anime and I love to languages, so this combination created a necesity inside of me to build an app to teach myself Japanese. It all started as a small personal proyect, but then it escalated and now this App is available on the Google Playstore.


Building Kanjizy has been a complete journey for me. I learnt how to implement and use so many things in Flutter. I struggled a lot with the implementation of the Flash Cards game section and Kanjizy's backend; dealing with multiple Animation Controlers and make the game function correctly, read multiple CSV files, write to those CSV files and make the app interact with the database was a complete odyssey!


I encorage you to learn new languages, and why not starting to learn Japanese with Kanjizy! Kanjizy uses my language learning method, which allows you to easly learn a big amount of vocabulary, verbs and words. What are you waiting for? Just click the button below and get started today!
